# Generative AI-Driven Reconstruction of Cultural Heritage

This repository provides reproducibility materials for the paper.

The repository includes:

manually curated knowledge-base materials;
the MLLM-based historical plausibility review prompt;
Blender-based mesh inspection scripts.

The multi-view generation prompts and photorealistic translation prompts are reported in the main text of the paper and are therefore not duplicated here.

Because the workflow uses third-party generative AI models, exact image and mesh outputs may vary due to model updates, random sampling, and platform settings. This repository is intended to support procedural transparency and technical verification rather than pixel-identical or geometry-identical reproduction.
